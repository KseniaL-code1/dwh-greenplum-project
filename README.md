# Разработка DWH и аналитической витрины продаж

## Описание
Спроектировано хранилище данных и реализована аналитическая витрина для анализа продаж.

## Бизнес-цель
Обеспечить возможность анализа продаж по каналам, продуктам и филиалам для поддержки управленческих решений.

## Стек
- Greenplum
- SQL
- DWH

## Архитектура
RAW → ODS → DDS → Data Mart

## Что сделано
- разработана архитектура хранилища данных
- спроектированы концептуальная, логическая и физическая модели
- выполнена нормализация до 3НФ
- реализована Star Schema
- разработана витрина dm_sales_channel_daily
- реализована инкрементальная загрузка данных

## Выводы
- обеспечена консистентность и целостность данных
- ускорено выполнение аналитических запросов за счет витрин
- подготовлены данные для BI-отчетности

## Результат
Реализована аналитическая витрина для ежедневного анализа продаж.
