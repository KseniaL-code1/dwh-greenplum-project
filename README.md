# Разработка DWH и аналитической витрины

## Описание
Спроектировано хранилище данных и реализована аналитическая витрина для анализа продаж.

## Стек
- Greenplum
- SQL
- DWH

## Архитектура
RAW → ODS → DDS → Data Mart

## Что сделано
- разработка моделей данных (концептуальная, логическая, физическая)
- нормализация до 3НФ
- построение Star Schema
- разработка витрины dm_sales_channel_daily
- реализация инкрементальной загрузки

## Результат
Обеспечена аналитика продаж по каналам, продуктам и филиалам.
